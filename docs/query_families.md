## Usage Notes
- PubMed and Europe PMC are used for high-precision biomedical harvesting.
- OpenAlex and CrossRef are used for broad scholarly graph expansion.
- Semantic Scholar is used sparingly for citation chasing on high-priority DOI seeds.
- ClinicalTrials.gov is harvested separately using the v2 structured API.
